

A simple and professional **Student Registration System** built using **C# Windows Forms (WinForms)**.  
This project is designed to practice and demonstrate key UI controls and fundamentals of desktop application development.

# 📖 Project Idea

The system allows users to register student information such as:

- Full Name  
- Email Address  
- Phone Number  
- Gender  
- Skills  
- Academic Level  

Then displays the collected data in a structured format using MessageBox or UI controls.

---

# ✨ Features

✔ Student data entry form  
✔ Gender selection using RadioButtons  
✔ Skills selection using CheckBoxes  
✔ Student level selection using Panel  
✔ Display data using MessageBox  
✔ Clear all fields functionality  
✔ About window using a separate form  
✔ GitHub LinkLabel integration  
✔ Clean and user-friendly UI design  

---

# 🛠️ Technologies Used

- C# (.NET Framework)
- Windows Forms (WinForms)
- Visual Studio
- Object-Oriented Programming (OOP Basics)
- UI Controls (Button, TextBox, Label, GroupBox, Panel, etc.)
